Política de Privacidade

Última atualização: 17 de julho de 2026

Esta Política descreve como este aplicativo ("o App") coleta, usa e protege informações ao integrar com a API do TikTok para agendamento e publicação de conteúdo de vídeo.

01 Informações que coletamos

O App coleta apenas as informações necessárias para publicar conteúdo de vídeo em nome do usuário autenticado, incluindo:

  • Token de acesso OAuth fornecido pelo TikTok após autorização explícita do usuário;
  • Identificador público da conta TikTok conectada;
  • Metadados dos vídeos enviados (título, legenda, hashtags).

02 Como usamos as informações

As informações coletadas são usadas exclusivamente para:

  • Autenticar o usuário junto à API do TikTok;
  • Publicar ou agendar vídeos autorizados pelo próprio usuário;
  • Manter o histórico interno de publicações para controle do usuário.

Não vendemos, alugamos ou compartilhamos essas informações com terceiros para fins de publicidade.

03 Armazenamento e segurança

Os tokens de acesso são armazenados de forma local e criptografada, utilizados somente para comunicação direta com a API oficial do TikTok. O usuário pode revogar o acesso a qualquer momento diretamente nas configurações da sua conta TikTok.

04 Retenção de dados

Os dados são mantidos apenas enquanto a integração estiver ativa. Ao revogar a autorização, os tokens associados são invalidados e removidos.

05 Direitos do usuário

O usuário pode, a qualquer momento, solicitar a exclusão de seus dados armazenados ou revogar o acesso do App à sua conta TikTok.

06 Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato através do e-mail informado no cadastro do aplicativo.
